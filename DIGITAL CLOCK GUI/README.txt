| DIGITAL CLOCK GUI in Python using Tkinter |

DIGITAL CLOCK GUI is a project for beginner.
This clock is implemented with the help of tkinter library and datetime library.

Tkinter Module:
Tkinter is the standard GUI library for Python. Python when combined with Tkinter provides a fast
and easy way to create GUI applications. Tkinter provides a powerful object-oriented interface to
the Tk GUI toolkit.

Datetime Module:
In Python, date and time are not a data type of its own, but a module named datetime can be imported
to work with the date as well as time. Datetime module comes built into Python, so there is no need 
to install it externally. 

Requirements: 
-python datetime library(inbuilted)
-python tkinter library(inbuilted)
-python 3

How to run:
-Just run the code in any command line
